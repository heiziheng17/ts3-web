TeamSpeak 3 使用指南

1

下载并安装客户端

访问 TeamSpeak 官网 下载对应系统的客户端，安装后启动。

支持 Windows、macOS、Linux、Android、iOS。
2

连接服务器

有两种方式连接：

方式一：点击页面顶部的 启动客户端 按钮，自动唤起客户端连接。

方式二：点击 复制地址 按钮，然后在客户端中：

1. 打开 TeamSpeak 客户端
2. 点击菜单 连接 → 连接
3. 在"服务器地址"粘贴复制的地址
4. 点击"连接"
3

设置昵称

首次连接会提示输入昵称，也可以在客户端设置中修改默认昵称：

设置 → 身份 → 默认昵称
4

切换频道

双击左侧频道列表中的频道名称即可切换。部分频道需要密码才能进入。

5

语音设置

建议配置麦克风和扬声器以获得最佳体验：

设置 → 捕获 → 选择麦克风设备
设置 → 回放 → 选择扬声器/耳机设备
设置 → 捕获 → 设置按键说话（推荐）
推荐使用"按键说话"模式，避免背景噪音干扰他人。
6

常用快捷键

Ctrl+Shift+M → 静音/取消静音
Ctrl+Shift+D → 禁用/启用麦克风
Ctrl+Shift+S → 开关声音
7

看不懂英文？

下载并安装简体中文语言包：

下载汉化补丁

Windows/macOS：双击 Chinese_Translation_zh-CN.ts3_translation 文件安装

安装后重启客户端，切换语言：

设置 → Options → Design → Language → 简体中文
来源：GitHub 项目

常见问题

Q: 连接不上服务器？

A: 检查网络是否正常，确认服务器地址和端口正确。部分公司/学校网络可能屏蔽了 TeamSpeak 端口。

Q: 别人听不到我说话？

A: 检查麦克风是否被系统静音，确认客户端中选择了正确的录音设备，检查是否设置了正确的按键。

Q: 有回声或杂音？

A: 在设置中开启回声消除和降噪功能，或使用耳机代替扬声器。
